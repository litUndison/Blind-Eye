Развлекательное приложение разработанное на Unity. В приложении присутствует сохранение всех гаданий текущей сессии, возможность настройки громкости звука и музыки, и сам процесс гадания. Гадание начинается с ввода вопроса, далее при загрузке сцены нужно выбрать 5 случайных карт из колоды(которые лежат рубашкой вверх). После недолгой загрузки пользователю выводится ответ и кнопка возвращения в меню. Само гадание реализовано через запрос к Gemini 2.5 flash. Нейросети передаётся значение выбранных карт и она генерирует красочный ответ на их основе.
